# Storybook Art

The React app expects these raster image files:

- `page-01.png`
- `page-02.png`
- `page-03.png`
- `page-04.png`
- `page-05.png`
- `page-06.png`
- `page-07.png`
- `page-08.png`
- `page-09.png`

Generate them with:

```bash
npm run imagegen:storybook
npm run imagegen:publish
```

`npm run imagegen:storybook` requires `OPENAI_API_KEY`.
