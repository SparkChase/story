# Imagegen Prompt Set

Mode note: these prompts are prepared for the `imagegen` skill's default built-in image generation path. The current session does not expose the built-in `image_gen` tool, so these are ready to run once that tool is available, or after explicit confirmation for the CLI fallback.

## Shared Style

Use case: illustration-story  
Asset type: romantic interactive React story page illustration  
Style/medium: soft children's book illustration, painterly pastel texture, rounded shapes, cinematic storybook lighting  
Color palette: pink theme with rose, peach, warm gold, muted lavender, soft teal accents  
Composition/framing: wide 16:10 scene, clear central action, generous negative space, no text  
Constraints: the two main characters are a small gentle boy named Xiao Yang and a small bright-eyed girl named Lily; keep them cute, warm, and consistent; no watermark; no written words inside the image  
Avoid: photorealism, harsh shadows, scary mood, cluttered background
